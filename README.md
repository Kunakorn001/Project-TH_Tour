TH_TOUR - ระบบจองทัวร์ออนไลน์

========================================
📌 รายละเอียดโปรเจ็ค
========================================
TH_TOUR เป็นระบบเว็บไซต์สำหรับจองทัวร์ภายในประเทศ
พัฒนาด้วย Django Framework โดยรองรับทั้งฝั่งลูกค้าและผู้ดูแลระบบ (Admin)

----------------------------------------
🎯 ฟังก์ชันหลักของระบบ
----------------------------------------
1. สมัครสมาชิก / เข้าสู่ระบบ
2. เลือกทัวร์และรอบการเดินทาง
3. เลือกประเภทห้องพัก (กรณีมีที่พัก)
4. กรอกข้อมูลผู้เดินทาง
5. ยืนยันการจอง
6. ระบบหลังบ้านจัดการข้อมูลและยืนยันการจอง

----------------------------------------
👤 วิธีการใช้งาน (ลูกค้า)
----------------------------------------
1. เข้าเว็บไซต์
2. สมัครสมาชิก หรือ Login
3. เลือกทัวร์ที่ต้องการ
4. เลือกรอบการเดินทาง
5. เลือกประเภทห้องพัก (ถ้ามี)
6. กรอกข้อมูลผู้เดินทาง
7. กดยืนยันการจอง
8. รอเจ้าหน้าที่ติดต่อกลับทางอีเมล

----------------------------------------
🛠️ วิธีการใช้งาน (Admin)
----------------------------------------
1. Login เข้าสู่ระบบหลังบ้าน
2. เพิ่มข้อมูล:
   - ทัวร์
   - รอบการเดินทาง
   - ที่พัก
   - ห้องพัก
3. ตรวจสอบรายการจองจากลูกค้า
4. ติดต่อกลับลูกค้าผ่านอีเมล
5. ยืนยันการจองในระบบ

หมายเหตุ 
- ในกรณีต้องการเพิ่ม Bochua ในรายละเอียดทัวร์แต่ละแพ็คเกจให้นำไฟล์ PDF ตั้งชื่อให้ตรงกับ Slug ของแต่ละแพ็คเกจทัวร์แล้วนำไปใส่ใน main/Bochua ตรวจสอบได้จากการกดปุ่มดาวน์โหลดรายละเอียด ในหน้าดูรายละเอียดทัวร์

----------------------------------------
🗂️ โครงสร้างข้อมูลหลัก
----------------------------------------
- Customer (ลูกค้า)
- Booking (การจอง)
- BookingTraveler (ผู้เดินทาง)
- Tour (ทัวร์)
- Schedule (รอบเดินทาง)
- RoomType (ห้องพัก)
- Accommodation (ที่พัก)
- Vehicle (ยานพาหนะ)
- Guide (ไกด์)

----------------------------------------
⚙️ วิธีรันโปรเจ็ค
----------------------------------------
1. ติดตั้ง Python และ Django
2. สร้าง Virtual Environment
3. ติดตั้ง dependencies:
   pip install -r requirements.txt
4. ทำ migration:
   python manage.py makemigrations
   python manage.py migrate
5. รันเซิร์ฟเวอร์:
   python manage.py runserver
6. เข้าใช้งานที่:
   http://127.0.0.1:8000
